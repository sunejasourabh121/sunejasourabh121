## 🚀 Featured Project

### [Real-Time ASL Sign Language Translator](https://github.com/sunejasourabh121/Sign_language_translator)
> **Patent Disclosure Pending — April 2026**

Offline, edge-deployed ASL fingerspelling translator running on a Raspberry Pi 4 at 15–20 fps — no GPU, no internet, hardware cost under ₹5,000.

- ~98% validation accuracy across 29 gesture classes
- Random Forest classifier trained on 87,000 images
- CLAHE preprocessing + wrist-relative landmark normalisation
- Built for India's ~18 million deaf and hard-of-hearing population
